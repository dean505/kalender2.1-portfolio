# Kalender 2.1 Frontend

React frontend for the Kalender 2.1 appointment booking application.

## Scripts

```bash
npm install
npm start
npm run build
```

## Configuration

Create a local `.env` file from `.env.example` when you need to override the backend URL:

```env
REACT_APP_API_BASE=http://localhost:8080
```

The local `.env` file is intentionally ignored by Git.
